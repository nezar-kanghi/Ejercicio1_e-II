En este ejercicio se nos pide realizar un programa que se conecte con base de datos y muestre por consola todos los ciclistas y que al mismo tiempo nos permita filtrar por pais del equipo (El que introduzca el usuario). para esto primero pediremos al usuario si quiere filtrar por pais, si su respuesta es si entonces se le pedira que escriba el pais por consola. Una vez optenidos los datos se realizara una consulta SQL para obtener los datos de todos los ciclistas WHERE pais = ? ( el introducido por el usuario ), uluego mostramos los datos mediante ResultSet, si la respuesta era no filtrar por pais se realizara lo mismo pero quitando el WHERE de la consulta SQL.
